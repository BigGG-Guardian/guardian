控制台使用方法:

            
您可以在浏览器控制台中直接调用以下方法生成签名:

            
// 生成签名
const sign = generateSignFromConsole({
  secretKey: "your-secret-key",
  algorithm: "hmac_sha256",
  queryParams: {},
  body: {
    name: "guardian",
    value: "test123"
  }
});

// 输出结果
console.log("签名:", sign.sign);
console.log("时间戳:", sign.timestamp);
console.log("签名字符串:", sign.signatureStr);
